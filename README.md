# Ford GoBike Sharing - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the 2018 Ford GoBike Sharing dataset to understand customer behavior, ride patterns, and operational trends. The analysis includes data cleaning, feature engineering, statistical analysis, and visualizations to generate actionable business insights for improving bike-sharing services.

---

## Objectives

- Analyze customer ride behavior
- Identify peak usage hours and days
- Study ride duration patterns
- Explore demographic trends
- Compare Subscribers and Customers
- Generate business recommendations

---

## 📂 Dataset

**Dataset:** Ford GoBike Trip Data (2018)

The dataset contains monthly bike trip records with information such as:

- Ride Duration
- Start & End Time
- Start & End Stations
- Bike ID
- User Type
- Member Gender
- Member Birth Year
- Bike Share Program

Total Records: **1.86+ Million**

## Key Insights

- Most rides last less than 15 minutes.
- Subscribers contribute the majority of rides.
- Peak demand occurs during weekday commuting hours.
- The 25–34 age group forms the largest user base.
- Male riders account for a higher proportion of trips.
- Ride durations tend to be longer on weekends.
- Extreme ride durations exist but represent only a small fraction of the data.

---

## Business Recommendations

- Increase bike availability during morning and evening peak hours.
- Expand docking stations in high-demand locations.
- Encourage Customers to become Subscribers through promotional offers.
- Improve awareness of the Bike Share for All program.
- Optimize bike redistribution using hourly demand patterns.
